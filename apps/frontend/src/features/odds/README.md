# ODDS Feature

- Nama resmi: One Dashboard Design System.
- Page route: `/odds`.
- API prefix: `/api/v1/odds`.
- Status: active.
- Ownership: intake, brief, queue, workflow, review, revision, escalation, reporting, ranking, dan konfigurasi ODDS.

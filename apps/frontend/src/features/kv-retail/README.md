# KV Retail Task Feature

- Page route: `/kv-retail`.
- API prefix: `/api/v1/kv-retail`.
- Status: active.
- Ownership: task list, assignee, workflow status, performance, option, dan file task KV Retail.

Feature ini tidak boleh memakai nama `Homework` atau mengimpor internal ODDS, Creative Report, atau Generator.

Public API module: `api/index.ts` (`kvRetailApi`).

# Primary Action Link

**Level:** Atom / Action  
**Application owner:** Core  
**Status:** review  
**Live documentation:** `/docs?section=components/primary-action-link`

`PrimaryActionLink` is a reusable primary navigation action. It uses link semantics and must not be used for form submission.

## Visual source

- Mobile source of truth: Figma file `TcuMZMQLJVhxq6KV75XK8g`, node `26:238`.
- Mobile size: 48px high with 4px outer padding and 36px radius.
- Label: Google Sans Flex Medium, 16px/20px, `#BA0DCB`, left padding 36px and right padding 16px.
- Arrow: 40px circular magenta asset with white forward arrow.
- Desktop keeps the same anatomy at 56px high, 18px/24px label, and 48px arrow.

Public props: `href`, `children`, `className`.
